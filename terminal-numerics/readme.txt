---------------------------------------------
	Terminal Numerics Install Information
---------------------------------------------

This software requires Java Runtime Environment (JRE) 1.8 or above. (http://java.com/download)
Tested primarily on Windows 7 64-bit. If you are using a different operating system, you may experience issues with the software. 

To run the software, double-click on 'Terminal Numerics.exe' or 'terminal-numerics.jar,' depending on your operating system.

Make sure you run the launcher in the same directory as the 'data' folder.
To select a custom 'data' folder, provide a command argument designating the location of the settings file within the desired directory.

If you encounter a problem with the software, please notify the developers.
